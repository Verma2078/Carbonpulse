In today’s computing environment, educational institutions and small
enterprises increasingly require centralized monitoring, energy-optimized
operations, and seamless communication between users and administrators.
To address this need, we developed a lightweight system that provides realtime tracking of CPU, GPU, RAM, and disk usage across multiple client
machines. It helps administrators efficiently monitor performance, manage
tasks, and communicate instantly with users.
The tool uses a client-server architecture, collecting system metrics every
second and displaying them through customizable visualizations. It features
a remote task manager for monitoring and terminating user-level
applications, a built-in live chat module for quick troubleshooting, and a
Carbon Pulse component that estimates power consumption and carbon
emissions per device. The result is a scalable, modular prototype that
integrates system diagnostics, sustainability, and effective communication.
